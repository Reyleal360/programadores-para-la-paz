# 🌐 Voz Ciudadana - Módulo 2 Día 5

Este proyecto es una plataforma de participación ciudadana que permite a la comunidad enviar propuestas y visualizarlas en tiempo real.

## 🚀 Cómo empezar

1. Asegúrate de estar en la carpeta `Modulo2/Semana1/Dia5`.
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia la API:
   ```bash
   npm run prueba
   ```
4. Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## 🛠️ Tecnologías utilizadas

- **Node.js + Express**: Servidor backend y API.
- **FS (File System)**: Persistencia de datos en archivos JSON.
- **Vanilla JavaScript**: Lógica del cliente usando Promesas (`.then`, `.catch`).
- **CSS (Glassmorphism)**: Interfaz de usuario moderna y responsiva.

## 📁 Estructura del proyecto

- `server.js`: Servidor Express y definición de rutas de la API.
- `data/propuestas.json`: Archivo donde se almacenan las propuestas.
- `public/`: Archivos estáticos del cliente (HTML, CSS integrado, JS).
- `semana4/`: Documentación y reflexiones de la clase.

## 📢 Objetivos de aprendizaje

- Comprender y manejar **Promesas** en JavaScript.
- Implementar comunicación cliente-servidor mediante **fetch**.
- Validar datos en el servidor y manejar errores de forma responsable.
- Documentar pruebas funcionales y procesos de depuración.
