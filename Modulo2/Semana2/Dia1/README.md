# 🏘️ Mini-Sitio Comunitario

Este es el proyecto único que trabajaremos durante toda la Semana 5 (Clases 21 a 25).

## 🚀 Cómo iniciar
1. Instala las dependencias: `npm install`
2. Inicia el servidor: `npm run sitio`
3. Abre [http://localhost:3000](http://localhost:3000)

## 📌 Flujo de trabajo en GitHub
Este es un **Proyecto Vivo**. La rama `main` será actualizada por los docentes con nuevas funcionalidades.
- Debes trabajar en tu propia rama: `git checkout -b nombres_apellidos`
- Cada día al iniciar, actualiza tu rama:
  ```bash
  git checkout nombres_apellidos
  git pull origin main
  ```

## 🛠️ Tecnologías
- **Node.js + Express**: API de servicios.
- **JavaScript Moderno**: Uso de `async` y `await` para el consumo de datos.
- **JSON**: Persistencia de mensajes y calendario editorial.
